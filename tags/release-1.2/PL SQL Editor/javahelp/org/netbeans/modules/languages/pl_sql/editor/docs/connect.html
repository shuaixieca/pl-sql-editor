Connecting to Oracle

First you need to register (add) new Oracle connection. Simply select Explorer's root node () and click Add. You should fill the Connection dialog with appropriate data. Current version of PL/SQL Editor use Oracle Thin JDBC Driver Version 11.1.0.7.0 to connect to Oracle DB. This driver is able to connect Oracle databases up 9i Release 2 (9.2.0.x).

Important: Saving user password is unsecure. Passwords are saved unencrypted!

In the Connection dialog you could check if the connection is working by pressing Test button. After the connection is saved you could find it in the Explorer.
