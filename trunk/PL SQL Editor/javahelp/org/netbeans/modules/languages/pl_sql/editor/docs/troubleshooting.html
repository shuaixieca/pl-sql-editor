Troubleshooting

  • To parse PL/SQL code first time PL/SQL Language module required about 500MB heap memory free. If memory is not enough the Out of Memory exception is occurred.

    Solution: Set “ -J-Xmx512m” option for launcher modifying /etc/*.conf file.

  • Sometimes PL/SQL Language module report an error in valid PL/SQL code.

    Solution: Please, create a new Issue on project homepage. Thank you!
