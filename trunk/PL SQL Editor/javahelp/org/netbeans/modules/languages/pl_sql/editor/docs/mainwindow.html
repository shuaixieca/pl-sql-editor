Main Window

The PL/SQL Editor main window consists of two parts:

  • Toolbar
  • Browser
The Toolbar is context sensitive and has the following functionality:
Icon Action
Add (create) corresponding new object under currently selected.
Delete (drop) currently selected object.
Refresh (reload) tree from currently selected object.
Select all objects. All objects from dba_objects view.
Select users own objects and granted objects. All objects from all_objects view.
Select only users own objects. All objects from user_objects view. Default
